I created a Daily Goal Tracker. 

It's a Chrome extension that turns your New Tab into a habit tracker. It's a visual reminder every time you open a new tab. As you mark each day complete, you see how much progress you are making toward one goal.

It's for someone that is interested in building a new habit. For me, it was to start writing HTML and CSS every day. After 3 weeks of building momentum for the new habit, I built this Chrome extension. Now I have a new goal to write every day for a couple of weeks.

### Things I learned
+ It's relatively simple to build a Chrome extension. If you can write some HTML and CSS, you're good to go.
+ `LocalStorage` is a feasible way to add persistance to your app without learning about databases yet. 
- I'd like to experiment with the Chrome bookmark API next time.
